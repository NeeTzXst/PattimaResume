Frontend / Backend / Flutter / Dart / Design

Software Engineering Project ( EVIE )

This project aims to develop an application called Evie. It's an app designed to facilitate the reservation of electric car chargers and parking spots for electric vehicles. The application starts by identifying pain points for electric car users and then involves stakeholder preparation and project management through the Scrum process. Station owners utilizing our platform can mark the locations of their stations on the map. Electric car users can utilize this app to reserve chargers and parking spots within a single platform, eliminating the need to use multiple apps for reservations.
